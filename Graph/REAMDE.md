## Graph (C++)
This folder contains implementations of basic graph algorithms using C++. The programs focus on fundamental traversal techniques and graph representation using an adjacency matrix.

## Topics Covered
* Breadth-First Search (BFS)
* Depth-First Search (DFS)
* Graph representation (Adjacency Matrix)
* Queue implementation (for BFS)
* Recursion (for DFS)
  
## Purpose
To build a strong foundation in graph traversal techniques and understand how graphs are explored programmatically.

## Technologies Used
C++

## Key Learnings
* Difference between BFS and DFS
* Level-order vs depth-wise traversal
* Importance of visited array to avoid cycles
* Practical use of queue and recursion
