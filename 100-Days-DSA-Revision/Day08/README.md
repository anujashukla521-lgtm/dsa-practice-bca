# Day 08 – String Basics

## Overview

Day 08 of my 100-Day DSA Revision Series focused on **String Basics**, one of the most fundamental topics in Data Structures and Algorithms. The goal was to understand string traversal, manipulation, character comparison, and frequency counting using basic techniques before moving on to advanced string algorithms.

---

## Topics Covered

* Introduction to Strings
* String Traversal
* Character Access using Indexing
* Two-Pointer Technique on Strings
* Palindrome Checking
* String Reversal
* Character Frequency Counting
* Time and Space Complexity Analysis

---

## Programs Implemented

### 1. `palindrome_check.cpp`

**Concepts Used:**

* Strings
* Two Pointers
* Character Comparison

**Description:**
Checks whether a given string is a palindrome by comparing characters from both ends and moving inward.

---

### 2. `reverse_string.cpp`

**Concepts Used:**

* Strings
* Two Pointers
* Swapping

**Description:**
Reverses a string without using any built-in reverse function by swapping characters from both ends.

---

### 3. `character_frequency.cpp`

**Concepts Used:**

* Strings
* Nested Loops
* Frequency Counting

**Description:**
Counts and displays the frequency of each unique character in a string while avoiding duplicate outputs.

## Key Learnings

* Strings can be efficiently processed using the two-pointer technique.
* Palindrome checking can be performed by comparing characters from both ends of the string.
* String reversal can be implemented using simple swapping operations.
* Character frequencies can be calculated using nested loops before introducing hashing techniques.
* Duplicate frequency outputs can be avoided by checking previously processed characters.
* Understanding basic string operations is essential before learning advanced string algorithms.
