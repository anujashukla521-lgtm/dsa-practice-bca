# Day 10 - Palindrome Problems

## Overview

Day 10 of the 100-Day DSA Revision Series focused on Palindrome Problems. The objective was to strengthen understanding of the Two Pointers technique and apply it to different palindrome-based problem variations. This included both basic palindrome validation and an advanced version allowing at most one character deletion.

---

## Topics Covered

* Palindrome String
* Two Pointers Technique
* Valid Palindrome II
* Case-Insensitive Character Comparison
* Alphanumeric Character Handling
* Helper Functions
* String Traversal and Validation

---

## Programs Implemented

### 1. palindrome_string.cpp

**Concepts Used:**

* Two Pointers
* String Traversal
* Palindrome Validation

**Description:**
Checks whether a given string is a palindrome by comparing characters from both ends and moving toward the center.

---

### 2. valid_palindrome_ii.cpp

**Concepts Used:**

* Two Pointers
* Helper Functions
* Conditional Character Deletion
* Case-Insensitive Comparison
* Alphanumeric Character Validation

**Description:**
Determines whether a string can become a palindrome after deleting at most one character. Uses a helper function to validate palindrome ranges when a mismatch occurs.
