# 100-Day DSA Revision Series

## Day 15 - Revision Day

### Overview

Day 15 was dedicated to revision and concept reinforcement. Instead of learning new topics or implementing new programs, the focus was on revisiting previously learned concepts using active recall, dry runs, and handwritten notes.

### Topics Revised

* Arrays
* Prefix Sum
* Range Sum Query
* Fixed Sliding Window
* String Concepts

### Revision Method

* Reviewed previously solved problems
* Performed dry runs on paper
* Recalled approaches without looking at solutions
* Identified concepts that require additional practice
* Focused on understanding patterns rather than memorizing code

### Programs Reviewed

* Array-based problems
* Prefix Sum implementations
* Range Sum Query problems
* Fixed Sliding Window problems
* String manipulation and pattern-based problems

### Complexity Analysis Practiced

* Array Traversal: O(n)
* Prefix Sum Construction: O(n)
* Range Sum Query: O(1) after preprocessing
* Fixed Sliding Window: O(n)
* Common String Operations: O(n)

### Key Learnings

* Revision is as important as learning new concepts.
* Remembering problem-solving patterns is more valuable than memorizing code.
* Active recall helps identify weak areas effectively.
* Consistent revision improves long-term retention.

### Day Outcome

* Strengthened understanding of previously learned concepts.
* Identified weak areas for focused practice.
* Reinforced problem-solving approaches through handwritten dry runs.
* Prepared for weak-area revision and practice in the next session.
