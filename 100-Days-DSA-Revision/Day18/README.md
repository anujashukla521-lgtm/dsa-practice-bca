# Day 18 - Medium Array Problems

## Overview

Day 18 focused on solving medium-level array problems using the Two Pointer technique. The goal was to strengthen pattern recognition skills and understand when to apply different variations of the two-pointer approach.

## Topics Covered

* Two Pointers (Same Direction)
* Two Pointers (Opposite Direction)
* Pattern Recognition in Array Problems
* Dry Run and Problem Analysis

## Programs Implemented

### 1. Container With Most Water

**File:** `container_with_most_water.cpp`

**Concepts Used:**

* Two Pointers (Opposite Direction)
* Greedy Pointer Movement
* Area Calculation
* Time Complexity Optimization

**Approach:**

* Start with pointers at both ends of the array.
* Calculate the container area formed by the selected heights.
* Move the pointer with the smaller height.
* Track the maximum area obtained.

### 2. Move Zeroes

**File:** `move_zeroes.cpp`

**Concepts Used:**

* Two Pointers (Same Direction)
* In-Place Array Modification
* Array Traversal

**Approach:**

* Use a fast pointer to scan the array.
* Use a slow pointer to place non-zero elements.
* Fill the remaining positions with zeroes.

## Complexity Analysis Practiced

### Container With Most Water

* Time Complexity: O(n)
* Space Complexity: O(1)

### Move Zeroes

* Time Complexity: O(n)
* Space Complexity: O(1)
