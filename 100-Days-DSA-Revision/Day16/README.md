## Programs Implemented

### 1. Maximum Average Subarray of Size K
**File Name:** `maximum_average_subarray.cpp`

**Concepts Used:**
- Sliding Window
- Array Traversal
- Optimization using fixed window size

**Approach:**
- Calculated the sum of the first window.
- Used sliding window technique to update the sum by removing the outgoing element and adding the incoming element.
- Compared window sums instead of averages because the window size remains constant.
