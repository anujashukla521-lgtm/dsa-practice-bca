# Day 02 - Arrays Basics | 100-Day DSA Revision Series

## Overview

Day 02 focused on strengthening the fundamentals of Arrays through basic problem-solving exercises. The goal was to understand array traversal, element comparison, counting techniques, and performing calculations on array elements. Along with coding, special attention was given to dry running algorithms on paper to improve logical thinking and debugging skills.

---

## Topics Covered

* Array Basics
* Second Largest Element in an Array
* Second Smallest Element in an Array
* Counting Even and Odd Elements
* Sum of Array Elements
* Average of Array Elements
* Dry Running and Tracing Array Algorithms
