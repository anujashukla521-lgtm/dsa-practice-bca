## 🎯 Key Concepts Learned

### Floor
- Finds the **greatest element** that is **less than or equal to** the target.

### Ceil
- Finds the **smallest element** that is **greater than or equal to** the target.

### Lower Bound
- Returns the index of the **first element greater than or equal to** the target.

### Upper Bound
- Returns the index of the **first element strictly greater than** the target.

### Search Insert Position
- Finds the index of the target if present.
- Otherwise, returns the position where the target should be inserted while maintaining sorted order.
