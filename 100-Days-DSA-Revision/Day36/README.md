# Day 36 - Recursion Basics

## 📌 Overview
Day 36 focused on understanding the fundamentals of recursion in C++. The goal was to build a strong foundation by solving basic recursive problems and understanding recursive function calls, base cases, and recursive thinking.

---

## 📚 Topics Covered

- Recursion Basics
- Base Case
- Recursive Calls
- Stack Unwinding
- Printing Patterns using Recursion
- Sum of First N Natural Numbers
- Power of a Number using Recursion

---

## 💻 Problems Implemented

1. Print Numbers from 1 to N
2. Print Numbers from N to 1
3. Print Even Numbers
4. Print Odd Numbers
5. Sum of First N Natural Numbers
6. Power of a Number

---

## 🧠 Key Learnings

- Understood how recursive functions call themselves.
- Learned the importance of defining a correct base case.
- Practiced breaking a problem into smaller subproblems.
- Learned the difference between recursive calls and returning values.
- Improved recursive thinking by tracing function execution.

---

## ⏱️ Time Complexity

| Problem | Time Complexity | Space Complexity |
|---------|-----------------|------------------|

| Print Even Numbers | O(N) | O(N) |
| Print Odd Numbers | O(N) | O(N) |
