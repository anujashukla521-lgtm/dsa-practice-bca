# Day 37 – Recursive Arrays

## 📌 Overview

Day 37 focused on solving array-based problems using recursion. The objective was to understand how recursion can be applied to traverse arrays, search for elements, validate array properties, and compute results without using iterative loops.

## 📚 Topics Covered

* Print array recursively
* Print array in reverse recursively
* Find first occurrence of an element
* Find last occurrence of an element
* Calculate sum of array elements recursively

## 💻 Problems Implemented

* Print Array
* Print Reverse Array
* First Occurrence
* Last Occurrence
* Sum of Array

## 🎯 Key Concepts Learned

* Traversing arrays using recursion
* Designing effective base cases
* Moving forward and backward through recursive calls
* Returning values from recursive functions
* Solving searching problems recursively
* Building confidence with recursion fundamentals before advanced topics

This session strengthened my understanding of recursion on arrays by implementing common recursive operations such as traversal, searching, validation, and aggregation. These problems helped build a solid foundation for more advanced recursion topics like strings, subsequences, and backtracking.
