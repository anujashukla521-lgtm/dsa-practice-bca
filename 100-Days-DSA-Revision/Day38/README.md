# Day 38 – Recursive Strings

## 📌 Overview

Day 38 focused on solving string-based problems using recursion. The goal was to strengthen recursive thinking by breaking a string problem into smaller subproblems until reaching the base case.

## 📚 Topics Covered

* Recursive string traversal
* Count characters recursively
* Count vowels recursively
* Check palindrome using recursion
* Reverse a string using recursion
* Remove all occurrences of a character recursively

## 💻 Problems Implemented

* String Traversal
* Count Characters
* Count Vowels
* Check Palindrome
* Reverse String
* Remove All Occurrences of a Character

## 🎯 Key Concepts Learned

* Defining an appropriate base case
* Processing one character at a time
* Returning values from recursive calls
* Building a new string recursively
* Understanding how recursion moves forward and backtracks through the call stack
* Solving string problems without using iterative loops
