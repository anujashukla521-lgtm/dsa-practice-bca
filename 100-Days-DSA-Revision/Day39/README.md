- Introduction to Recursion
- Understanding Recursive Function Calls

## 🎯 Purpose

The purpose of this day was to build a strong foundation in recursion by solving basic problems.

The focus was on understanding:
- How recursive calls work
- How problems can be broken into smaller subproblems
- Importance of base conditions
- Returning values through recursive calls

---

## 💻 Problems Implemented

### 1. Factorial using Recursion

- Calculated factorial of a number using recursive approach.
- Learned how a problem reduces until reaching the base case.

### 2. Fibonacci Series using Recursion

- Implemented Fibonacci sequence recursively.
- Understood overlapping recursive calls.

### 3. Find Maximum Element in Array

- Used recursion to traverse an array.
- Compared elements recursively to find the maximum value.

### 4. Find Minimum Element in Array

- Implemented recursive array traversal to find the smallest element.

### 5. GCD using Recursion

- Implemented Euclidean Algorithm recursively.
- Learned how recursive mathematical relations simplify problems.

### 6. Linear Search using Recursion

- Performed searching in an array using recursive calls.
- Practiced recursion with index-based traversal.

### 7. Check if Array is Sorted

- Used recursion to check whether array elements are arranged in increasing order.

## 🧠 Key Learnings

- Every recursive function requires a proper base case.
- Recursive calls solve smaller versions of the same problem.
- Returning values correctly is important in recursion.
- Array problems can be solved by reducing the problem size using indexes.
- Recursion is the foundation for advanced topics like Backtracking and Dynamic Programming.
