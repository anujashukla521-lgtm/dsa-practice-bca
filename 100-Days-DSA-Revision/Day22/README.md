# Day 22 – Binary Search

## 📖 Overview

Day 22 of my **100-Day DSA Revision Series** was dedicated to learning the fundamentals of **Binary Search**. I revised how Binary Search works, its prerequisites, and implemented its basic variations to build a strong foundation for more advanced Binary Search problems.

---

## 📚 Topics Covered

* Introduction to Binary Search
* Prerequisites for Binary Search
* Iterative Binary Search
* Binary Search on Descending Sorted Array
* Time and Space Complexity

---

## 💻 Programs Implemented

| File Name                      | Concept                                              |
| ------------------------------ | ---------------------------------------------------- |
| `binary_iterative_search.cpp`  | Iterative Binary Search on an ascending sorted array |
| `binary_search_descending.cpp` | Binary Search on a descending sorted array           |

---

## ⚡ Complexity Analysis Practiced

| Algorithm                        | Time Complexity | Space Complexity |
| -------------------------------- | --------------: | ---------------: |
| Iterative Binary Search          |    **O(log n)** |         **O(1)** |
| Binary Search (Descending Array) |    **O(log n)** |         **O(1)** |

---

## 🎯 Key Learnings

* Understood why Binary Search works only on sorted data.
* Learned how Binary Search reduces the search space by half in every iteration.
* Implemented Binary Search iteratively using the two-pointer approach (`low` and `high`).
* Learned how the comparison logic changes when searching in a descending sorted array.
* Revised the importance of calculating the middle index safely to avoid overflow.

---

## ✅ Day Outcome

Built a solid understanding of the Binary Search algorithm by implementing its iterative version and adapting it to work with descending sorted arrays. This forms the foundation for solving more advanced Binary Search interview problems.
