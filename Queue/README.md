# Queue Implementation in C++

This repository contains implementations of the **Queue data structure** using different approaches in C++. It is part of my Data Structures practice and focuses on understanding core concepts through hands-on coding.

##  Topics Covered

* Queue using Array
* Queue using Linked List
* Circular Queue
* Basic Queue Operations:
     * Enqueue (Insertion)
     * Dequeue (Deletion)
     * Peek / Front
     * Check Empty / Full
  
##  Purpose

The goal of this repository is to:
* Strengthen understanding of Queue concepts
* Practice implementation from scratch (without STL)
* Improve problem-solving skills for interviews and exams

## Technologies Used

* Language: C++
* Concepts: Data Structures, Pointers, Dynamic Memory Allocation

##  Key Learnings

* Difference between linear and circular queues
* Efficient memory usage in circular queues
* Dynamic memory handling using pointers
* Real-world applications of queues (e.g., scheduling, buffering)

##  Future Improvements

* Implement Queue using STL for comparison
* Add more advanced problems (e.g., Queue using two stacks)
* Optimize code and handle edge cases
