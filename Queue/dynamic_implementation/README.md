# Dynamic Queue Implementation in C++

This folder contains the implementation of a **Queue using a Linked List**, allowing dynamic memory allocation. Unlike static queues, this implementation can grow and shrink during runtime.

##  Topics Covered

* Queue using Linked List (Dynamic Allocation)
* Node-based Data Structure
* Queue Operations:
  * Enqueue (Insertion at rear)
  * Dequeue (Deletion from front)
  * Peek / Front
  * Empty Queue Handling

##  Purpose

The purpose of this implementation is to:
* Understand dynamic memory allocation in queues
* Overcome limitations of static queues
* Learn pointer-based data structure design
* Build interview-level understanding of queue implementation

##  Technologies Used

* Language: C++
* Concepts: Pointers, Dynamic Memory Allocation, Linked Lists

##  Key Learnings

* Efficient memory utilization with dynamic allocation
* No fixed size limitation
* Importance of maintaining `front` and `rear` pointers
* Real-world usability compared to static queues

##  Considerations

* Requires proper memory management (avoid memory leaks)
* Slightly more complex than array-based implementation
* Extra memory used for pointer storage

##  Future Improvements

* Implement Queue using two stacks
* Add destructor for memory cleanup
* Compare with STL queue for performance and readability
