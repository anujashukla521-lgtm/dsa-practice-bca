# Static Queue Implementation in C++

This folder contains the implementation of a **Queue using a static array**, where the size is fixed at compile time. It demonstrates how queues work under memory constraints and helps build a strong foundation of linear data structures.

##  Topics Covered

* Queue using Array (Static Allocation)
* Linear Queue Concept
* Queue Operations:
  * Enqueue (Insertion)
  * Dequeue (Deletion)
  * Peek / Front
  * Overflow and Underflow Handling

##  Purpose

The purpose of this implementation is to:
* Understand how queues behave with fixed memory allocation
* Learn limitations of linear queues (like memory wastage)
* Build logic from scratch without using STL
* Strengthen problem-solving for exams and technical interviews

##  Technologies Used

* Language: C++
* Concepts: Arrays, Index Management, Conditional Logic

##  Key Learnings

* Fixed-size data structures and their constraints
* Concept of **overflow** when queue is full
* Inefficiency of linear queues due to unused space
* Importance of circular queues as an improvement

##  Limitations

* Size is fixed and cannot grow dynamically
* Memory wastage after multiple dequeue operations
* Not suitable for real-world scalable applications

##  Future Improvements

* Implement Circular Queue to overcome memory issues
* Compare performance with dynamic queue
* Add edge case handling and validations
