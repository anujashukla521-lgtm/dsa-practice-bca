### Linked List - DSA (C++)

## Topics Covered
 * Singly Linked List
 * Node Creation & Linking
 * Traversal of Linked List
 * Finding Middle Element (Fast & Slow Pointer)
 * Reversing Linked List (Iterative Approach)

## Purpose

This folder contains implementations of fundamental linked list operations in C++. The goal is to build a strong understanding of pointer manipulation and efficient traversal techniques, which are essential for both academic exams and technical interviews.

## Technologies Used 
 * C++
 * Standard Template Library (basic usage)
 * Pointer-based data structures
   
## Key Learnings
 * Understanding how nodes are connected using pointers
 * Using slow and fast pointer technique to solve problems efficiently
 * Reversing a linked list in-place (O(1) space)
 * Importance of maintaining links to avoid losing data
 * Writing clean and modular functions
   
## Future Improvements
 * Implement insertion and deletion operations
 * Doubly and Circular Linked List implementations
