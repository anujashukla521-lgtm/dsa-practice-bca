# DSA Revision – Day 1 Programs (C++)

This section contains basic C++ programs practiced during my DSA revision journey. These programs focus on strengthening core programming concepts such as functions, loops, bitwise operations, and problem-solving fundamentals.

---

## 1. Even or Odd Checker (Using Bitwise Operator)

### 📌 Concepts Used:
- Bitwise AND operator (`&`)
- Binary representation of numbers
- User-defined functions
- Conditional statements (`if-else`)
- Basic input/output in C++

### 🧠 Description:
This program checks whether a given number is even or odd using the bitwise AND operator. It evaluates the least significant bit of the number (`n & 1`) to determine the result.

## 2. Multiplication Table Generator

### 📌 Concepts Used:
- Functions in C++
- `for` loop iteration
- Parameter passing in functions
- Basic input/output handling
- Formatted output

### 🧠 Description:
This program generates the multiplication table of a given number from 1 to 10 using a user-defined function and a loop for iteration.
