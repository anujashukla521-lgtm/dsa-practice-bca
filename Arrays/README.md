# Array Operations in C++
This repository contains basic C++ programs that demonstrate different **array operations and searching techniques** used in Data Structures.

## Files Included
- Array_ADI.cpp** – Implementation of Array ADT (Abstract Data Type).
- Array_insertion.cpp** – Program to insert an element into an array.
- array_deletion.cpp** – Program to delete an element from an array.
- arr_insertion_deletion.cpp** – Demonstrates both insertion and deletion operations.
- arr_linear_search.cpp** – Implementation of Linear Search to find an element in an array.
- arr_binary_search.cpp** – Implementation of Binary Search for searching elements in a sorted array.

## Concepts Covered

- Array ADT
- Array Insertion
- Array Deletion
- Linear Search
- Binary Search

These programs are written in **C++** and are useful for understanding the **basic array operations in Data Structures**.
